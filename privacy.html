개인정보처리방침

최종 수정일: 2025년 1월

1. 수집하는 정보

본 사이트는 회원가입을 요구하지 않으며, 별도의 개인정보를 직접 수집하지 않습니다. 다만 아래와 같은 정보가 간접적으로 처리될 수 있습니다.

  • 테마 설정: 다크/라이트 모드 선호 설정이 브라우저의 localStorage에 저장됩니다. 이 정보는 사용자의 기기에만 저장되며 외부로 전송되지 않습니다.
  • 생년월일: 전생 테스트, 오늘의 운세 등에서 입력하는 생년월일 정보는 브라우저 내에서만 처리되며, 서버로 전송되거나 저장되지 않습니다.
  • 문의 양식: 비즈니스 문의 시 입력한 이름, 이메일, 회사명, 문의 내용은 Formspree 서비스를 통해 전달됩니다.

2. 사진 및 이미지 처리

얼굴 분석, 퍼스널 칼라 진단, 얼굴 나이 테스트 등에서 업로드하는 사진은 사용자의 브라우저에서만 처리됩니다. face-api.js 라이브러리가 브라우저 내에서 직접 분석을 수행하며, 사진은 어떠한 서버로도 전송되지 않고, 외부에 저장되지 않습니다.

심리 테스트(연애 유형, 인간관계 유형 등)의 응답 결과도 브라우저 내에서만 처리되며, 별도로 저장되지 않습니다.

3. 제3자 서비스

본 사이트는 아래 제3자 서비스를 사용합니다. 각 서비스의 개인정보 처리에 대해서는 해당 서비스의 정책을 참조해 주세요.

  • Disqus - 댓글 기능 제공. Disqus 자체적으로 쿠키를 설정할 수 있습니다.
  • Formspree - 문의 양식 처리.
  • Google AdSense - 광고 표시를 위해 쿠키를 사용할 수 있습니다.
  • jsDelivr CDN - face-api.js 라이브러리 및 모델 파일 제공.

4. 쿠키

본 사이트 자체적으로는 쿠키를 설정하지 않습니다. 다만, 위에 명시된 제3자 서비스(Disqus, Google AdSense 등)가 자체적으로 쿠키를 사용할 수 있습니다.

5. 데이터 보관

사용자의 테스트 결과, 업로드 사진 등은 브라우저 세션 내에서만 존재하며, 페이지를 떠나면 자동으로 삭제됩니다. 서버에 저장되는 데이터는 없습니다.

6. 문의

개인정보 처리와 관련된 문의 사항은 각 페이지 하단의 문의 양식을 통해 연락해 주세요.
